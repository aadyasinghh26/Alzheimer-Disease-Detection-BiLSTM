# Early Alzheimer’s Disease Detection using ML and BiLSTM

This project detects early Alzheimer’s disease using clinical biomarkers from the OASIS dataset.

Models used:
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- BiLSTM

Techniques:
- Data preprocessing
- SMOTE class balancing
- Feature scaling
- Deep learning using BiLSTM

Dataset:
OASIS Longitudinal Dataset
https://www.oasis-brains.org/
